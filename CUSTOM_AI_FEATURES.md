# KaTrain カスタムオプション・AI戦略設定 詳解ドキュメント

本ドキュメントでは、コミット履歴および最新のコードをもとに実装された**「Human-like（人間らしい）モードの追加オプション機能」**や、**新たなAI戦略「力戦派（Fighting）」**の挙動ロジック、および各種設定項目について、内部の仕組みを知らない方でも理解できるようにわかりやすく解説します。

---

## 1. Human-like モードの基本改良：二段階フィルタ（悪手フィルタ）

### 概要
従来のHumanモードは「人間の棋譜をもとに予測された手（人間が打ちそうな手）」を優先して打つ仕組みでした。しかし、これだけだと「プロでも稀に打つ大悪手（ポカ）」までAIが再現してしまい、突然自滅してしまう課題がありました。
これを防ぐため、**「人間が打ちそうな手の中から、実際に致命的な損にならない手だけを選び出す仕組み（悪手フィルタ）」**が採用されています。

### 挙動の仕組み（二段階のクエリ評価）
1. **Stage 1 (人間の手探し):** AIが「人間ならどこに打つか？」という候補と優先順位をリストアップします。
2. **Stage 2 (正確な損得計算):** リストアップされた候補手に対し、「実際にその手を打つと何目損するか」をバイアスなしの正確なAI分析で再度チェック（クリーンクエリ）します。
3. **フィルタリング:** その際、設定された「許容される損失（目数）」を超える大悪手は、いくら人間らしくても候補から除外されます。

**【進行度に応じた閾値（どれくらい損しても許すかのライン）】**
* **序盤:** 布石の段階では、少しのミスも致命傷になりやすいため判定を厳しく設定しています。（例: 19路盤なら約2.8目以上の損を許さない）
* **中盤・終盤:** 局面が複雑になると「多少の損」は人間らしさとして許容するようになります。（例: 13/19路盤なら約6.0目までの損を許す）
* ※9路盤や13路盤では、盤面が狭いため、より厳しい（損を許容しない）評価設定になります。

## 2. 発展機能：第一感ブレ（First Impression Deviation）

### 概要
「人間はぱっと見の第一感で、最善手とは別の『ちょっとだけ損な手』を選んでしまうことがある」という特性を再現するためのオプションです。以前は一部の領域に限定された「大差勝ちフィルター」という仕組みがありましたが、現在はこれを廃止し、**全盤面に対応した「第一感ブレ」機能**として統合・拡張されています。

### 挙動の仕組み
* **わざと少し損な手を探す:** 人間が選びそうな手が複数ある場合、上位3候補の中で**「最善ではないが、大悪手でもない手（0.5目〜最大2.0目程度の損）」**をあえて選ぶことで、「いかにも人間らしいミスやつまづき」を誘発します。

### Green Blend (グリーンブレンド) 機能
もし第一感（一番人間が打ちそうな手）が、0.5目未満の「ほぼ損のない良い手（緑色の枠で表示されるような手）」だった場合、**「そのままその緑の手を打つか、あえてブレた手（0.5〜2.0目損）を探すか」**を確率でランダムにばらけさせる機能です。
これにより、AIの打ち回しが単調にならず、プレイするたびに異なる自然な人間らしさを表現します。

## 3. 安全機能と補正ロジック

### 安全弁（Safety Valve）とクロスチェック
AIが考えた「人間らしい手」が、**どうしても4.0目以上の大損になってしまう場合**に作動します。
いくら人間が選びそうだからといって、それを選ぶとゲームが壊れてしまうため、強制的に「一番スコアの良い最善手」に軌道修正します。最近のアップデートにより、この安全弁が正しく作動するよう「通常分析とのクロスバリデーション（本当に安全な手かどうかの二重チェック）」が組み込まれ、見かけ倒しの手に騙される事故を防いでいます。

### 拮抗タイブレーク（Close-call Tiebreak）
二つの候補手が「どちらも同じくらい評価が高い（拮抗している）」と迷ったときの決定ルールです。
以下の状態になったとき、AIは**「どうせ人間らしさが同じくらいなら、実際に2目以上スコアが得になる手を選ぶ」**よう補正機能が働きます。
* 第一候補と第二候補の人間らしさ確率がほぼ同じ（5%以内の差）。
* 探索深さが逆転している（AIの内部の読み（MCTS）で候補の順位が揺れている）。

## 4. AI戦略設定：力戦派（Fighting）のロジック

### 概要
大人しく陣地を囲うのではなく、**「相手の石に接近して戦いにつなげる」「まだ境界線がはっきりしていない場所（未確定領域）に飛び込む」**ことを好む、新しいAIプレイスタイルです。

### 挙動の仕組み
力戦派のAIは、手を決める際に以下の**「力戦重み（Fighting Weight）」**を追加で考慮し、評価を上乗せします。
1. **未確定度 (Unsettledness):** すでに誰の陣地か確定している場所ではなく、まだ勝負がついていない（お互いに陣地にできそうな白黒つかない）場所を高く評価します。
2. **相手石への接近 (Proximity):** 相手の石との距離を計算し、相手の石に近い（くっつく、プレッシャーをかける）場所ほど高く評価します。

これらをかけ合わせることで、**「勝負がついておらず、かつ相手の石の近く（＝ドンパチ起きそうな場所）」**へ好んで打ち込むようになります。

## 5. 各種設定項目・パラメータの分かりやすい解説

設定画面等で調整可能な主なパラメータと、それがどう影響するかの説明です。

| パラメータ名 / 概念 | 解説 |
| :--- | :--- |
| **第一感ブレ<br>(First Impression Deviation)** | オンにすると、AIが「ぱっと見の直感」を信じて、少しだけポイントを下げる（0.5〜2.0目損の）手を時々選ぶようになります。非常に人間臭いプレイスタイルになります。 |
| **序盤の第一感ブレ<br>(First Impression Opening)** | 布石（序盤）の段階でも第一感ブレを有効にする設定です。序盤のミスは致命的になることがあるため、盤石な立ち上がりをさせたい場合はオフにしますが、より人間らしさを出したい場合はオンにします。 |
| **Green Blend<br>(first_impression_green_blend)** | 第一感が「少しだけ惜しい良い手（損が0.5目未満）」だった時、それを優先するか、もう少しブレた他の手を探すかを柔軟に決定する機能です。 |
| **Green Ratio<br>(green_blend_green_ratio)** | 上記Green Blendの中で、「少しだけ惜しい良い手（緑枠の手）」をそのまま素直に打つ確率（0.0〜1.0）です。高い数値ほど、大きなミス（ブレ）をせず手堅くなります。 |
| **Threshold (悪手フィルタ基準)** | AIが「これはさすがに酷いミスだから打つのをやめよう」と判断するボーダーラインです。内部的に序盤は約2.8目、中終盤は6.0目の損で線引きされています（13/19路盤基準）。 |
| **力戦モード<br>(Fighting Mode)** | AIの思考プロセスを切り替えます。<br>・**ScoreLoss:** 得失点をベースに乱戦を好むモード。<br>・**Human:** 人間の打ち方をベースに乱戦を好むモード（9段程度の実力に設定）。<br>・**Classic:** 通常AIベースの荒削りなモード。 |
| **力戦最大損失<br>(Fighting Max Loss)** | 力戦派の「ScoreLossモード」時に影響する、「戦いを起こすためならどれくらい（目数）損を妥協できるか」の限界値（デフォルト3.0目）。この数字を大きくすると、少々無理をしてでも殴り込みに行きます。 |
| **Unsettled Power / Proximity Stddev** | （力戦派の内部パラメータ）「どれくらい未確定の場所を好むか」「相手の石のどれくらい近くを好むか」の度合いです。これらの数値が高いほど、より激しいインファイト（接近戦）になります。 |
| **Endgame Threshold<br>(endgame)** | 力戦派（や他の特殊スタイル）の**「こだわりを捨てるタイミング」**となる進行度の割合です。（デフォルト0.45）。<br>19路盤で `0.45` の場合、162手目を過ぎると特殊な手選びを停止し、通常のヨセに移行します。これを設定しないと終盤に暴発するため重要な安全装置になります。<br>※注意: 力戦派のモードが「Human」の場合はこの設定値は**無視**され、内部固定の「0.50（進行度50%）」の専用設計が強制的に優先適用されます。 |
